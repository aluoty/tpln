# tpln
This is tpln, a lightweight TUI planner built using ncurses and C.
Run by:
```bash
make
./build/tpln
```
Clean by:
```bash
make clean
```
## Controls
+ j and k key to move through tasks
+ e for drag and drop in manual order mode
+ s to cycle sort types
+ a to add tasks
+ d to delete task
+ SPACE to toggle the completion of task
+ q to exit
